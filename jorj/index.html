Jewel or Juice UI

Project Details

This project is a UI design for an existing business in Hawaii called Jewel or Juice. The business specializes in selling smoothies, coffee, and acai bowls. This project was a re-design of the existing website.

Process

The process involved 3 rounds of UI designs using Adobe XD. After exploring 3 rounds of different designs, one design was chosen to be finalized. The challenging part of this project was designing the menu pages. Since Jewel or Juice has such an extensive menu, it was difficult coming up with an effective way to display all of the items.
